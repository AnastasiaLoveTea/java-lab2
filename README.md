## Опис проєкту

Проєкт реалізує завдання лабораторної роботи №2 з дисципліни «Java / Технології програмування».  
Доменна область – хтонічні істоти (нечисть).  
Програма демонструє використання Stream API в поєднанні з власним `Gatherer` та `Collector` для генерації, фільтрації, групування та статистичного аналізу даних.

Основні кроки, які виконує застосунок:

1. Створює нескінченний генератор об’єктів `ChthonicCreature` з випадковими (але реалістичними) параметрами.
2. За допомогою власного `Gatherer` відкидає перші *N* істот певного виду (Поле A) та формує список із 500 елементів.
3. Фільтрує список за діапазоном Параметра Б (кількість років з моменту першої згадки) та групує об’єкти за Полем В (вид істоти).
4. Збирає статистику за Полем Г (сила атаки) у клас `AttackStats` через власний `Collector`: обчислюються мінімум, максимум, середнє значення та стандартне відхилення.
5. Аналізує значення сили атаки: обчислює міжквартильний розмах, визначає викиди та формує структуру вигляду `{data: X, outliers: Y}`.

## Доменна область

Хтонічна істота описується класом `ChthonicCreature` з такими полями:

- `name` – ім’я істоти;
- `species` – вид істоти (Поле A / Поле В);
- `firstMention` – дата першої згадки в літературі;
- `attackPower` – сила атаки (Поле Г).

## Використані технології

- Java 25 (OpenJDK 25)
- Stream API
- `java.util.stream.Gatherer` (JDK 24+)
- Maven для збірки проєкту

## Вимоги до оточення

- Встановлена JDK 25 або новіша (з підтримкою `Stream.gather`).
- Встановлений Maven 3.x.
- Операційна система: Windows / Linux / macOS.

## Інструкція зі збірки та запуску

1. Клонувати репозиторій:

   ```bash
   git clone <url-репозиторію>
   cd lab2
   
2. Зібрати проєкт за допомогою Maven:

mvn clean package

3. Запустити застосунок одним із способів:

через IDE (IntelliJ IDEA): запустити метод main у класі ua.lab2.Lab2StreamApp;

4. Результати роботи (групування за видами, статистика сили атаки, кількість викидів) будуть виведені в консоль.
